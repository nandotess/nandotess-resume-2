## Development

### Install NPM
https://nodejs.org/en/

### Install NPM modules
`npm install`

### Gulp Commands
* `gulp init` shortcode for `gulp watch`
* `gulp watch` watch for changes on files to recompile them
* `gulp build` build JavaScript and Sass files
